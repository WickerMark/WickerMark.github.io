Air Resistance and Varying Air Density with Height

Setting Initial Parameters

To begin the new section of code, we must initialise some new variables in regards to the ISA model and calculating varying air density at height.

The inputs to the model are the same as the model with air resistance except for two. The input of air density has been removed and assumed to be 1.225 \(kgm^{-3}\) for the linear air density model because the air density will be constantly updating in the new model. The acceleration due to gravity has been set to 9.81 \(ms^{-2}\) since the model is only valid for Earthâ€™s atmosphere.

Also, there are some conditions which must be set. These are variables which are defined before the Verlet integration and are passed into the function which performs the calculations. These are sea level standard atmospheric pressure (\(P_{0}\)), specific gas content (\(R\)), the mass of one air molecule (\(m\)), the Boltzmann constant (\(k_{B}\)) and the standard temperature at sea level (\(T_{0}\)).

\(P_{0}=101325\) Pa

\(R=287.052874\) \(Jkg^{-1}K^{-1}\)

\(m=4.81\times10^{-26}\) kg

\(k_{B}=1.380649\times 10^{-23}\) \(JK^{-1}\)

\(T_{0}=15^{\circ}C\)

Verlet Integration

For a greater understanding of how Verlet Integration works, refer to my Challenge 9 post.

The function is identical to the one used in the air resistance model, bar the addition of four new calculations.

Also, notice that the calculation for the air resistance factor is now located within the Verlet integration function. This ensures that the air density and air resistance factor are changing with every iteration. The current y coordinate is used to update the height of the model. This is so the air resistance factor is adjusted accordingly at each height.

Then, the x and y coordinate of the projectile are updated as with Challenge 9.

The four new equations are:

\(T_{h}\approx T_{0}-h\times0.00650\)

\(T_{Kelvin}=T_{Celsius}+273.15\)

\(P_{h}=P_{0}e^{\frac{-mgh}{k_{B}T_{h}}}\)

\(\rho_{h}=\frac{P_{h}}{RT_{h}}\)

The first equation approximates the temperature at a given height. This temperature is converted to Kelvin. The third equation calculates air pressure at height, \(h\), given the Boltzmann constant, the temperature at heigh, \(h\), the pressure at sea level, the mass of an air molecule and the acceleration due to gravity. This is then used to calculate air density at height, \(h\), using the fourth equation.

# Verlet method for varying air density
def verlet_air(p0, h0, sgc, mass_am, kB, T0, d_x, d_y, ux, uy, m, A, dt, c_drag, u, h, density):
    t = 0
    x_air_plot = []
    y_air_plot = []

    x_air_plot.append(d_x)
    y_air_plot.append(d_y)
    
    n = 0
    while y_air_plot[n] >= 0:
        k = (0.5 * c_drag * density * A) / m

        ax = - (ux / u) * k * u**2
        ay = - g - (uy / u) * k * u**2
        d_x = d_x + ux * dt + 0.5 * ax * dt**2
        d_y = d_y + uy * dt + 0.5 * ay * dt**2
        x_air_plot.append(d_x)
        y_air_plot.append(d_y)
        ux = ux + ax * dt
        uy = uy + ay * dt
        u = math.sqrt(ux**2 + uy**2)
        h = d_y

        temp = T0 - h * 0.00650
        temp = temp + 273.15
        p = p0 * math.exp((-mass_am * g * h) / (kB * temp))
        density = p / (sgc * temp)

        n += 1

        t = t + dt

    return x_air_plot, y_air_plot, n, t

x_air_plot, y_air_plot, n, t_air = verlet_air(p0, h0, sgc, mass_am, kB, T0, x_air, y_air, ux, uy, m, A, dt, c_drag, u, h, density)

Time of Flight and Range

The time of flight is calculated by incrementing the value of t by the timestep throughout the iterations of the Verlet integration. This is output, along with a phrase.

The range is calculated by considering the last x coordinate calculated, since this is the last instant when \(y\geq 0\).
